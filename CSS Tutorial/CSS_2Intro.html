CSS(Cascading Style Sheet) Intro2

Element Specific Style Class

An element specific style class starts with the element name, followed by a dot operator, which is followed by the class name.

Syntax:
element_name.class_name{class definition}
h1.red{color:red;}

Using Two or More classes

CSS syntax:
p.red{color:red;}//This attribute is applied to a class having the word "red" in the p tag. It means, it is applied to two p tag below in the html code. This is an element specific style.
p.look{font-size:1.2em;}
//This attribute is applied to a class having the word "look" in the p tag. It means, it is applied to only one p tag below in the html code. This is an element specific style.
.align{text-align:center;}//This attribute is applied to a class having the word "align" in the p and h1 tag. It means, it is applied to only one p and one h1 tag below in the html code.This is a universal style
